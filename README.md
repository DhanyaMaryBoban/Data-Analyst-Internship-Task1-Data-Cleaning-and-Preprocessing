# Data-Analyst-Internship-Task1-Data-Cleaning-and-Preprocessing

## 📌 Objective
Clean and prepare a raw dataset by handling missing values, duplicates, inconsistent formats, and incorrect data types. This ensures the dataset is structured and ready for analysis.

## 📂 Dataset
The Mall Customer Segmentation dataset contains customer information including:
- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

## 🛠️ Tools Used
- **Python (Pandas, NumPy)**  

## 🔎 Steps Performed
- Checked for **missing values** → none found  
- Removed **duplicate records** → none found  
- Standardized **data types** → numeric features validated, categorical variable (Gender) confirmed  
- Ensured **column headers** are clean and uniform  

## ✅ Deliverables
- Cleaned dataset (ready for analysis)  
- PDF from google colab of code and outputs that performed in Jupyter Notebook
- Short summary of changes 

## 🎯 Learning Outcomes
By completing this task:
- Gained hands-on experience in identifying and fixing common data issues  
- Learned to use **Pandas functions** like `.isnull().sum()`, `.drop_duplicates()`, and `.dtypes`  
- Improved understanding of **data preprocessing** as a critical step before analysis
